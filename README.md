# uTag
## a multi-format music tag management lib 一个支持多格式的.Net音乐标签管理库
**目前还在开发中**
 ---
### feature：
1.  无视音乐格式，统一管理标签。
2. 修改标签直接修改属性即可
3. 使用.Net Standard 1.0进行开发，理论上同时支持.Net Framework和.Net Core（UWP）

### 预计支持的格式：
 [x] id3v2.3(目前只读)
 [x] flac(目前只读)
 [ ] ogg
 [ ] ape
 [ ] id3v2.4
 [ ] m4a
 [ ] dsf
 [ ] aiff
